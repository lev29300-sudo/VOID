# VOID
[ENG] VOID is a low-level programming language designed to combine Python-like simplicity with powerful control over hardware. It provides direct, efficient management of GPU and RAM, giving you the performance of assembly with a much cleaner syntax.
VOID is an independent open-source project. If you appreciate the effort and want to help me accelerate development and dedicate more time to low-level optimization, you can support me with a donation. Every contribution helps with development tools, hosting, and resources.
